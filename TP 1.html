XML Schema

Definition

XML schema publie comme recommandation par le W3C en mai 2001 est un langage de description de format de document XTML permattant de definir la structure et le type de contenu XML. cette definition permet notamment de verifie la validite de ce document

Modele de XML Schema

la recommendation specifie des documents XML a partir dun modele abstrait elle en fournit le format XML pour ce modele abstrait un schema est un exemple de composants

Exemple 

 
< ?xml version= "1.0" encoding ="UTF-8"?>
<xs :schema xmlns : xs = "https://www.w3.org/2001XMLSchema">
<xs:element name="personne">
    <xs:complexType>
	   <xs:sequence>
	     <xs:element name="nom"type "xs:string "/>
         <xs:element name="prenom"type "xs:string "/>
		 <xs:element name="date_naissance"type "xs:date "/>
		 <xs:element name="etablissement"type "xs:string "/>
		 <xs:element name="num_tel"type "xs:string "/>

	    </xs:sequence>
	</xs:complexType>	
</xs:element>
</xs:schema >
